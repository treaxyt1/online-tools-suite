What is Open Graph?

The Open Graph protocol enables any web page to become a rich object in a social graph. For instance, this is real estate for Facebook, LinkedIn, and Pinterest.

By using Open Graph tags, you can control how your content is displayed when shared on social media. Without these tags, social networks will try to guess which title, description, and image to use, often with poor results.

Key Open Graph Tags:

  • og:title - The title of your object as it should appear within the graph.
  • og:type - The type of your object, e.g., "video.movie". Depending on the type you specify, other properties may also be required.
  • og:image - An image URL which should represent your object within the graph.
  • og:url - The canonical URL of your object that will be used as its permanent ID in the graph.
