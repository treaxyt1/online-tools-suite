About Hash Functions

Hash functions convert input data into a fixed-size string. They are commonly used for:

  • MD5: 128-bit, fast but not cryptographically secure
  • SHA1: 160-bit, deprecated for security use
  • SHA256: 256-bit, secure, widely used
  • SHA512: 512-bit, most secure
