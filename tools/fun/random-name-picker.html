How to Use the Random Name Picker

  1. Enter names in the text box (one name per line)
  2. Click "Spin the Wheel" to start the selection
  3. Watch the wheel spin and land on a random winner!
  4. Perfect for giveaways, raffles, team selection, and more

Pro tip: You can click directly on the wheel to spin it!
